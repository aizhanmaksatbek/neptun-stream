# neptun-stream
Blog for posting articles.
The system is written using the FastAPI, database, Python.

# Endpoints:
- Publish articles - POST
- Delete articles - DELETE
- Get articles by the filters - GET

# Database:
Store the articles


Create your virtual environment:
bash
```
python -m venv .venv_neptun
source .venv_neptun/bin/activate
deactivate
```
